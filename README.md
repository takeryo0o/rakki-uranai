# 今日のラッキー占い 名前入力・200文版

## 主な仕様

- 名前入力あり
- 占い項目は総合ラッキー度のみ
- ラッキーアイテム、カラー、数字、時間を表示
- 「今日のひとこと」は200種類
- 同じブラウザでは1日1回
- 名前に「遠藤」が含まれる場合、80％の確率でラッキー度が1～20

## 「遠藤」の判定例

低運勢処理の対象:
- 遠藤
- 遠藤さん
- 山田遠藤
- 遠藤太郎

対象外:
- えんどう
- Endo

## 今日のひとこと

`script.js` の `adviceList` に200文を直接登録しています。
占うと、その中から1文をランダムに選び、日付が変わるまで保存します。
